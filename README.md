# 💰 Expense Tracker

A full-stack Expense Tracker using Node.js, Express, MongoDB,

## Features
- Add / Edit / Delete expenses
- Category-based tracking
- Date tracking
- Total calculation
- Chart visualization

## Tech Stack
- Frontend: HTML, CSS, JS
- Backend: Node.js, Express
- Database: MongoDB

## Run
npm install  
npm run dev  

http://localhost:5000
